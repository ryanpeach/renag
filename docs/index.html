renag¶

A Regex based linter tool that works for any language and works exclusively with custom linting rules.

iregex¶

All regex captures in this module default to using ``iregex` <https://github.com/ryanpeach/iregex>`_. ``iregex` <https://github.com/ryanpeach/iregex>`_ can help make your regex more understandable to readers, and allow you to compose large regex statements (see examples/regex.py for examples). ``iregex` <https://github.com/ryanpeach/iregex>`_ defaults to multiline regex using the re module.
